# Paper Planner 📝

A beautiful, interactive to-do list and habit tracker inspired by physical paper planners. 

![Paper Planner Interface](screenshot.png)

## Features 🌟

- **Monthly Habit Tracker**: Keep track of your daily habits and protocols over the entire month.
- **Score Graph**: Visualize your daily habit consistency and progress with an interactive graph.
- **Daily Planner**: Organize your primary and secondary tasks for any selected day.
- **Learning Desk**: Create and manage topic pages for taking notes and structuring ideas (like the "What is GitHub?" example).
- **Realistic Paper Design**: Clean, notebook-style aesthetic with ring binders and lined paper elements.

## Getting Started 🚀

1. Clone this repository:
   ```bash
   git clone https://github.com/vk-harsh/To-Do-List.git
   ```
2. Navigate to the project directory:
   ```bash
   cd To-Do-List
   ```
3. Open `index.html` in your favorite web browser, or serve it using a local development server (e.g., `npx serve` or `python -m http.server`).
